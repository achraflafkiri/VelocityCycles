API Documentation

Get All Sales

Endpoint: 127.0.0.1:3000/api/v1/sales

Description: Get all sales with a maximum of 2 buyer cards.

    "status": "success",
    "date": {
        "length": 19,
        "sales": [

                "_id": "64b4271166eb613d09378a35",

                "buyer_card": [],
                "__v": 0,
                "id": "64b4271166eb613d09378a35"
            },
            // ... (other sales)
        ]
    }
}

Get One Sale

Endpoint: 127.0.0.1:3000/api/v1/sales/{sale_id}

Description: Get details of a specific sale.

{
    "status": "success",
    "data": {
        "tour": {
            "_id": "64b4271166eb613d09378a35",
            "name": "docker ",
            "price": 120,
            "date": "Wed Jan 01 2020 00:00:00 GMT+0100 (Central European Standard Time)",
            "registrationNumber": "2020/01/01",
            "buyer_firstname": "ahmed",
            "buyer_lastname": "ali",
            "buyer_card": [],
            "__v": 0,
            "id": "64b4271166eb613d09378a35"
        }
    }
}

Error Response (if sale not found):

{
    "status": "fail",
    "message": "no sale with this id"
}

Delete a Sale

Endpoint: 127.0.0.1:3000/api/v1/sales/{sale_id}

Description: Delete a specific sale.

Status Code: 204 (No Content)
